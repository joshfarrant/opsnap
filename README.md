# Op Snap — West Midlands

Parsing, geocoding, and visualising [West Midlands Police Operation Snap](https://www.westmidlands.police.uk/police-forces/west-midlands-police/areas/campaigns/campaigns/operation-snap/) data.

Operation Snap lets the public submit dashcam and photo evidence of driving offences. West Midlands Police publish monthly PDFs of all submissions and outcomes. This project turns those PDFs into structured, geocoded, queryable data.

## What's here

```
data/pdfs/          Monthly PDFs from WM Police (Jan 2025 – Feb 2026)
data/opsnap.db      SQLite database (generated by ingest.py)
scripts/ingest.py   Parse PDFs → SQLite
scripts/geocode.py  Geocode locations via Nominatim → lat/lng
metadata.json       Datasette config with canned queries
```

## Quick start

```bash
# Install dependencies
pip install pdfplumber

# Ingest PDFs into SQLite
python scripts/ingest.py

# Geocode locations (takes ~3 hours, cached, resumable)
python scripts/geocode.py

# Browse with Datasette
pip install datasette datasette-cluster-map
datasette -i data/opsnap.db -m metadata.json
```

## Data

~25,600 records across 14 months (Jan 2025 – Feb 2026). Each record contains:

| Field | Example |
|-------|---------|
| Reporter transport mode | Car, Pedestrian, Pedal Bike |
| Vehicle make/model/colour | BMW 3 Series Black |
| Offence | Driving without Due Care and Attention |
| Second offence | No Insurance |
| Location | Washwood Heath Road |
| Council area | Birmingham |
| Disposal | Fixed Penalty, Education, Warning Letter, Court, NFA |

## Key stats

- **71%** of submissions result in action (education, warning, fine, or court)
- **Washwood Heath Road** is the most-reported location (923 reports)
- **~25,600** total submissions across 14 months
- **Ford** is the most-reported vehicle make, followed by Volkswagen and Mercedes-Benz

## Datasette queries

The metadata.json includes canned queries for:

- Top offence locations (with map)
- Disposal breakdown by month
- Vehicle makes ranked
- Pavement parking hotspots (with map)
- Offence type breakdown
- Reporter type vs offence type
- Council area comparison
- All geocoded offences on a map

## Data source

Monthly PDFs from [West Midlands Police Operation Snap](https://www.westmidlands.police.uk/police-forces/west-midlands-police/areas/campaigns/campaigns/operation-snap/). Location data geocoded via [OpenStreetMap Nominatim](https://nominatim.openstreetmap.org/). Map tiles by [CARTO](https://carto.com/).
